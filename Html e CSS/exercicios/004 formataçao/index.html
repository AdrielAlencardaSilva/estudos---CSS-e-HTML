Negrito e itálico / formataçâo

Negrito / destaque

Essa frase está usando o termo negrito de forma (não semântica) utilizando a tag B.


Já nessa frase utiliza o termo em negrito de forma (semântica) utilizando a tag
STRONG.

Itálico / Ênfase

Essa frase está em itálico de forma (nâo semântica) utilizando a tag I.


Agora essa frase está em itálico de forma (semântica) utilizando a tag EM.

Texto subrescrito

Para inserir coisas do tipo x elevado a 10, utiliza a tag SUP. X10.

Texto subescrito

Texto subescrito é quando você quer colocar por exemplo, H2O no seu texto.Para isso se utiliza a tag SUB.Ficando assim H2O.

Palavra ou frase sublinhada

Para deixar uma palavra sublinhada é só utilizar a tag INS.

Texto marcado

Para marca uma palavra ou frase é só utilizar a tag MARk.

Texto grande e texto pequeno

Para deixa a palavra ou frase bem destacada é só utilizar a tag BIG.

Agora para deixar a palavra pequena é só utilizar a tag SMALL.

Palavra excluida

Para deixar é só utilizar a tag DEL.
